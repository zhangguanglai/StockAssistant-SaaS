# 财务面板优化方案

## 一、现状问题分析

### 1.1 视觉设计问题
- 颜色使用混乱：红/绿/黄/蓝混用，没有统一的语义
- 间距不一致：卡片内边距、元素间距不统一
- 字体层级不清：标题、数值、标签字体大小跳跃

### 1.2 信息层级问题
- 核心指标（PE/ROE/股息率）和次要指标（PB/毛利率/负债率）平铺展示
- 财务健康评分和维度明细占据左侧大量空间，但重要性不如核心指标
- 趋势图位置靠下，不够突出

### 1.3 交互体验问题
- 指标含义只有简单的 ⓘ 提示，不够直观
- 表格数据没有悬停高亮
- 缺乏数据对比（如同比、环比）

---

## 二、优化方案

### 2.1 视觉设计优化

#### 颜色系统统一
```css
/* 财务面板专用颜色系统 */
--finance-positive: #10b981;  /* 绿色 - 健康/优秀 */
--finance-warning: #f59e0b;   /* 黄色 - 注意/一般 */
--finance-danger: #ef4444;    /* 红色 - 风险/较差 */
--finance-neutral: #6b7280;   /* 灰色 - 中性/无数据 */
--finance-highlight: #3b82f6; /* 蓝色 - 重点/选中 */
```

#### 间距系统
```css
/* 统一间距 */
--finance-gap-xs: 4px;
--finance-gap-sm: 8px;
--finance-gap-md: 12px;
--finance-gap-lg: 16px;
--finance-gap-xl: 24px;
```

#### 字体层级
```css
/* 字体大小层级 */
--finance-title: 18px;      /* 面板标题 */
--finance-section: 14px;    /* 区块标题 */
--finance-value: 28px;      /* 核心数值 */
--finance-label: 12px;      /* 标签文字 */
--finance-caption: 11px;    /* 辅助说明 */
```

### 2.3 交互体验优化

#### 悬停提示增强
- 核心指标卡片悬停：显示指标计算公式和参考区间
- 表格行悬停：高亮整行，显示环比/同比变化
- 评分维度悬停：显示该维度的具体计算逻辑

#### 展开详情
- 核心指标点击展开：显示历史趋势迷你图
- 财务健康评分点击展开：显示4维度的详细评分说明

#### 动画效果
- 数值变化时：数字滚动动画
- 评分变化时：进度条平滑过渡
- Tab切换时：内容淡入淡出
