## 1. 产品概述

### 1.1 产品定位

面向个人 A股投资者的智能投研工具，提供**持仓管理、智能选股、策略复盘、K线分析、持仓建议**等全流程功能，帮助投资者提升决策效率。

### 1.2 核心价值矩阵

| 维度 | 描述 | 状态 |
|------|------|------|
| **选股引擎** | 三套策略（趋势突破/板块龙头/超跌反弹），6步筛选+100分评分 | ✅ 已完成 |
| **持仓管理** | 买入/卖出/编辑/删除，多用户隔离，SQLite 持久化 | ✅ 已完成 |
| **策略中心** | 市场环境判断、仓位建议、选股回测、实时预警 | ✅ 已完成 |
| **复盘体系** | 周/月/全周期胜率和盈亏比统计，策略效果跟踪闭环 | ✅ 已完成 |
| **技术分析** | K线图（日K）+ MA均线 + 成交量，支撑/压力位参考 | ✅ 已完成 |
| **市场数据** | 涨停/北向/龙虎榜/板块资金/板块轮动 | ✅ 已完成 |

### 2.2 文件结构

```
stock-portfolio/
├── server.py              # Flask 应用入口（~68行）
├── helpers.py             # 共享辅助函数（行情获取、缓存、计算）
├── auth.py                # JWT 认证模块
├── database.py            # SQLite 数据库操作
├── screener.py            # 选股引擎核心逻辑
├── config.py              # 配置文件
├── routes/
│   ├── auth_bp.py         # 认证路由（登录/注册/刷新）
│   ├── position_bp.py     # 持仓/交易/复盘/回测/预警
│   ├── screen_bp.py       # 选股引擎路由
│   ├── watch_bp.py        # 观察池/策略效果/股票对比
│   └── data_bp.py         # 市场数据（涨停/北向/龙虎榜等）
├── static/
│   ├── css/style.css      # 样式文件
│   └── js/app.js          # Vue 应用逻辑
├── templates/
│   └── index.html         # 主页面模板
├── data/                  # 数据文件（JSON缓存、数据库）
└── portfolio.db           # SQLite 主数据库
```

### 2.3 认证体系

- **JWT 双 Token**：access_token（72h）+ refresh_token（7天）
- **密码安全**：PBKDF2 哈希（10万次迭代）
- **多租户**：所有业务表含 user_id 外键
- **默认账号**：default / default123

## 3. 已实施功能

### 3.1 持仓管理模块 ✅

| 功能 | 说明 | API |
|------|------|-----|
| 持仓 CRUD | 创建/编辑/删除/清仓 | `GET/POST/PUT/DELETE /api/positions` |
| 交易记录 | 买入/卖出记录，支持部分卖出 | `POST /api/positions/{id}/sell` |
| 实时行情 | 新浪财经实时价格（东方财富降级） | `GET /api/quotes` |
| 资金管理 | 总资产/可用资金/持仓市值/浮动盈亏 | `GET /api/capital` |
| 资产分布 | 饼图展示（含现金扇区） | `GET /api/positions/summary` |
| 持仓建议 | 5维评分（±100分）→ 加仓/持有/观望/减仓/清仓 | `GET /api/positions/{id}/advice` |
| 止损止盈 | 6个支撑/压力位参考，点击自动填入 | `GET /api/positions/{id}/levels` |

### 3.2 选股引擎模块 ✅

#### 3.2.1 三套策略

| 策略 | 代码 | 适用环境 | 核心逻辑 |
|------|------|---------|---------|
| **趋势突破** | `trend_break` | 上升/震荡期 | MA20站稳 + MACD金叉 + 放量 + 板块效应 |
| **板块龙头** | `sector_leader` | 情绪亢奋期 | 板块涨>2% + 涨幅5-9.5% + 换手5-15% + 主力流入 |
| **超跌反弹** | `oversold_bounce` | 下跌末期 | 近20日跌>20% + 止跌信号 + 市值>100亿 |

#### 3.2.2 统一评分体系（0-100分）

| 维度 | 趋势突破 | 板块龙头 | 超跌反弹 |
|------|---------|---------|---------|
| 趋势/跌幅 | 0-40 | 涨换 0-40 | 跌幅/信号 0-55 |
| 板块 | 0-20 | 0-25 | 0-15 |
| 资金 | 0-20 | 0-20 | 0-15 |
| 加分 | 0-20 | 0-15 | 0-15 |

#### 3.2.3 策略参数可调

每个策略 5-7 个核心参数，运行前弹窗确认/修改。

#### 3.2.4 API

| 接口 | 说明 |
|------|------|
| `GET /api/screen/strategies` | 获取策略列表+智能推荐 |
| `GET /api/screen/params?strategy=xxx` | 获取策略可调参数 |
| `POST /api/screen/run` | 执行选股（支持 strategy + params）|
| `GET /api/screen/result?strategy=xxx` | 获取选股结果 |

### 3.7 数据源策略 ✅

#### 3.7.1 三级降级链

```
实时行情获取
  ├─ 东方财富接口（当前服务器不可访问）
  │    └─ 降级到新浪财经（可用，批量支持）
  │         └─ 降级到 Tushare 日线（历史数据）
  └─ 非交易时段 → 直接使用 Tushare
```

#### 3.7.2 缓存策略

| 数据类型 | 交易时段 TTL | 非交易时段 TTL |
|---------|-------------|---------------|
| 实时行情 | 30秒 | 5分钟 |
| 大盘指数 | 30秒 | 5分钟 |
