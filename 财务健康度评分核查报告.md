## 三、问题诊断

### 3.1 核心问题：运营效率指标选择不当

**当前设计：** 运营效率 = 净利率

**问题分析：**
1. **净利率属于盈利能力指标**，不应作为运营效率的衡量标准
2. **运营效率**应该衡量企业资产/资金的使用效率，常用指标包括：
   - 存货周转率 = 营业成本 / 平均存货
   - 应收账款周转率 = 营业收入 / 平均应收账款
   - 总资产周转率 = 营业收入 / 平均总资产
   - 营业周期 = 存货周转天数 + 应收账款周转天数

3. **商用车行业特性**：
   - 重卡行业净利率普遍偏低（5-10%）
   - 但存货周转快（订单式生产）
   - 应收账款周转受下游客户影响

### 3.2 评分标准过于严苛

| 指标 | 当前满分标准 | 行业实际情况 | 问题 |
|------|-------------|-------------|------|
| 毛利率 | 40% | 制造业平均20-30%，重卡约8-15% | 标准过高 |
| 净利率 | 20% | 制造业平均5-10% | 标准过高 |
| 负债率 | 30% | 制造业平均50-70% | 标准过低 |

### 3.3 中国重汽实际评分验证

根据截图数据：
- ROE: 10.6% → 12分
- 毛利率: 7.8% → 2分
- 净利率: 约5-8% → 3分
- 负债率: 65.3% → 5分
- 营收增长: 假设10% → 8分

**总分**: 12 + 2 + 3 + 5 + 8 = **30分**（与截图38分接近）

## 四、优化建议

### 4.1 运营效率指标重构

**方案A：使用周转率指标（推荐）**

```javascript
// 需要从Tushare获取额外数据
// - balancesheet: 存货(inventories)、应收账款(account_receiv)
// - income: 营业成本(oper_cost)、营业收入(revenue)

// 存货周转率 = 营业成本 / 平均存货
const inventoryTurnover = oper_cost / ((begin_inventory + end_inventory) / 2);

// 应收账款周转率 = 营业收入 / 平均应收账款  
const arTurnover = revenue / ((begin_ar + end_ar) / 2);

// 总资产周转率 = 营业收入 / 平均总资产
const assetTurnover = revenue / ((begin_assets + end_assets) / 2);

// 运营效率评分（25分）
const turnoverScore = 
  (inventoryTurnover >= 6 ? 10 : inventoryTurnover >= 3 ? 7 : inventoryTurnover >= 1 ? 4 : 0) +
  (arTurnover >= 8 ? 10 : arTurnover >= 4 ? 7 : arTurnover >= 2 ? 4 : 0) +
  (assetTurnover >= 1 ? 5 : assetTurnover >= 0.5 ? 3 : 0);
```

**方案B：简化版（使用现有数据）**

```javascript
// 如果无法获取周转率数据，可暂时使用ROA替代净利率
// ROA = 净利润 / 总资产，反映资产使用效率

const roa = parseFloat(fina.roa) || 0;

// 运营效率评分（25分）
const efficiencyScore = 
  roa >= 0.10 ? 25 :
  roa >= 0.07 ? 20 :
  roa >= 0.05 ? 15 :
  roa >= 0.03 ? 10 :
  roa >= 0.01 ? 5 : 0;
```

### 4.2 评分标准调整

**建议调整后的评分标准：**

| 指标 | 原标准 | 建议标准 | 理由 |
|------|--------|----------|------|
| 毛利率 | 40%满分 | 30%满分 | 符合制造业实际 |
| 净利率 | 20%满分 | 15%满分 | 符合制造业实际 |
| 负债率 | 30%满分 | 40%满分 | 制造业合理负债率 |
| ROE | 20%满分 | 15%满分 | 持续15%ROE已是优秀企业 |

### 4.3 行业差异化评分（进阶）

```javascript
// 根据行业类型调整评分标准
const industry = stockInfo.industry;

// 行业系数配置
const industryConfig = {
  '汽车整车': { marginFactor: 0.6, debtFactor: 1.2 },  // 低毛利、高负债正常
  '银行': { marginFactor: 1.5, debtFactor: 0.5 },      // 高杠杆是商业模式
  '白酒': { marginFactor: 1.0, debtFactor: 1.0 },      // 标准
  // ...
};

const config = industryConfig[industry] || { marginFactor: 1.0, debtFactor: 1.0 };

// 应用系数调整评分
const adjustedGrossScore = grossScore * config.marginFactor;
const adjustedSafetyScore = safetyScore * config.debtFactor;
```

## 五、实施建议
