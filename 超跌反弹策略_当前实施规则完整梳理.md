## 三、总分汇总公式

```
总分 = min(
    趋势强度(≤35) 
  + 板块效应(≤15) 
  + 资金面(≤15) 
  + 加分项(≤15) 
  + 技术改善(≤15) 
  + HL结构(-5~+10), 
  100
)
```

**理论上限**: 35+15+15+15+15+10 = **105** → 截断为 **100**
**理论下限**: 0+5+0+0+0+(-5) = **0** → 保底 **0**
