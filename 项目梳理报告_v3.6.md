## 一、项目定位与产品概览

### 1.1 产品定义
**A股个人投资助手** —— 一款面向 A 股散户的 **全栈 Web 应用**，集持仓管理、智能选股、策略分析、市场数据于一体。

### 1.2 核心价值主张
| 维度 | 描述 |
|------|------|
| **目标用户** | A股个人投资者（散户） |
| **解决痛点** | 选股无依据、持仓无复盘、数据分散 |
| **差异化** | 6步量化筛选 + 100分评分体系 + 智能预警 |
| **部署方式** | 本地 Flask 服务 / Railway 云部署 |

**文件规模**: ~2981 行（最大模块）  
**成熟度**: ⭐⭐⭐⭐ 高度成熟

| 交易记录管理 | `/api/positions/<id>/trades` | ✅ 完整 |
| 资金账本 | `GET/POST /api/capital` | ✅ 完整 |
| 大盘指数行情+结构分析 | `GET /api/index` | ✅ 完整 |
| K线分析(含前复权) | `GET /api/kline/<code>` | ✅ 完整 |
| 个股详情(财务+技术面) | `GET /api/stock/<code>/detail` | ✅ 完整 |

**亮点能力**:

- 价格锚点计算引擎（支撑/阻力/买入区间/目标位）

### 3.2 🔍 选股引擎模块 (`screener.py` + `routes/screen_bp.py`)

**文件规模**: ~3500 行（screener） + ~389 行（screen_bp）  
**成熟度**: ⭐⭐⭐⭐ 成熟，持续迭代中

**三套策略体系**:

| 策略 | 标识 | 核心逻辑 | 适用场景 |
|------|------|---------|---------|
| **趋势突破 v3.1** | `trend_break` | MA20>MA60 + MACD多头 + 量比>1.3 + 连续上涨 | 震荡偏强市 |
| **板块龙头** | `sector_leader` | 板块涨幅领先 + 成分股资金流入 + 换手率 | 结构性行情 |
| **超跌反弹 v3.3** | `oversold_bounce` | 近20日跌幅>20% + 止跌信号 + 低估值 | 下降趋势末端 |

**选股流程 (6步)**:
```
大盘环境判断(STEP 0)
  → 全市场初筛(STEP 1: ~5000只→~500只)
  → 技术指标过滤(STEP 2: MA/MACD/KDJ/量比)
  → 基本面评分(STEP 3: ROE/营收/PE/PB)
  → 板块热度加分(STEP 4: 概念匹配)
  → 最终排序(STEP 5: 100分综合排名)
```

**性能优化亮点**:
- 批量日线接口替代逐只查询：O(N×API) → O(days×API)
- 文件缓存机制（概念板/资金流/日线批量）
- 进程内缓存交易日历
- 异步线程执行选股（不阻塞Web请求）

## 四、helpers.py 公共工具层详解

这是整个系统的**基础设施层**，约 2000 行代码，包含以下子系统：

### 4.1 数据源降级链
```
新浪财经(15s TTL) → 东方财富(30s TTL) → Tushare日线(5min TTL)
```

### 4.2 缓存体系
| 类型 | 实现 | 用途 |
|------|------|------|
| 内存TTL缓存 | `cache_get/set/clear` | 行情/指数通用缓存 |
| 文件缓存 | `_save_cache/_load_cache` | 选股中间结果(概念板/资金流/日线) |
| 进程缓存 | `_stock_list_cache` | 股票列表(24h) |
| ST/停牌缓存 | `_st_stock_cache` | 当日ST/停牌集合 |

### 4.3 技术分析引擎
| 函数 | 能力 | 版本 |
|------|------|------|
| `analyze_hl_points()` | 高低点结构分析（HH/HL/LH/LL统计） | v3.5 统一重构 |
| `get_hl_structure()` | 趋势判定(上升/下降/震荡/底部/顶部) | v3.5 |
| `calc_atr_profile()` | ATR波动率分析(四档分类) | v4.2 P2-7 |
| `calc_price_time_prob()` | 价位到达概率/预期天数 | v4.2 P2-8 |
| `calc_price_hit_rate()` | 历史价位命中率回测 | v4.2 P2-9 |
| `calc_buy_price_anchors()` | 买入场景专用价格锚点 | P1-6 |
| `_generate_smart_alerts()` | 7维智能预警 | v3.4.1 |
