# A股短线选股规则分析 & 量化实施方案

> 从实战交易者视角，系统拆解规则合理性、盲区与落地路径

---

## 一、必选条件逐条拆解

### 1. 大盘处于上升或震荡阶段 ⭐⭐⭐⭐⭐

**合理性：高。** 大盘是最大的β因子。统计上看，上涨阶段个股上涨概率约65%，下跌阶段仅约35%。

**实战要点：**
- 「上升」容易量化，但「震荡」的定义是关键——是箱体震荡（上下界明确）还是横盘整理？
- 建议用**20日均线斜率 + 布林带宽度**双指标判断：
  - 上升：MA20斜率 > 0 且价格在MA20上方
  - 震荡：MA20斜率接近0（|斜率| < 阈值），布林带收窄
- **盲区：** 大盘可能盘中跳空，盘后判断滞后。建议用日线收盘确认，盘中做参考

**量化实现：**
```
指数MA20 = MA(收盘价, 20)
指数MA20斜率 = (今日MA20 - 5日前MA20) / 5日前MA20
大盘状态 = "上升" if 斜率 > 0.002
         = "震荡" if -0.002 < 斜率 <= 0.002
         = "下降" if 斜率 <= -0.002
```

---

### 2. 股价站稳20日均线 ⭐⭐⭐⭐

**合理性：较高。** MA20是中短期趋势的生命线，站上MA20说明中期趋势向上。

**实战要点：**
- 「站稳」不能只看当天，至少要连续2-3日在MA20上方才确认
- **关键细节：** 是「收盘价站稳」还是「最低价不破」？建议用收盘价 > MA20 × 1.01（留1%余量），过滤假突破
- 配合**MA20斜率向上**（斜率 > 0）效果更好，避免横盘MA20的假信号

**盲区：**
- 刚从下方突破MA20的第一天，可能是假突破（拉高出货），需要放量确认
- 建议加条件：**突破当日成交量 > 5日均量的1.3倍**

**量化实现：**
```
站稳条件 = 收盘价 > MA20 * 1.01
         AND 收盘价 > MA20（前1日）
         AND 收盘价 > MA20（前2日）
         AND MA20斜率 > 0
```

---

### 3. 所属板块今日涨幅 > 1% ⭐⭐⭐⭐

**合理性：高。** 板块效应是A股最强特征之一。板块涨 > 1% 说明有资金集中进攻。

**实战要点：**
- 板块数据来源：东方财富概念板块/行业板块实时排行
- **关键问题：** A股一只股票往往属于多个概念（如宁德时代同时属「新能源车」「储能」「锂电池」）。取哪个？
  - **建议：取该股所属的所有概念板块中涨幅最大的那个**
- 板块涨幅要考虑**跟风时间差**：领涨龙头启动后1-2小时，跟风股才有表现。如果盘中筛选，板块已涨 > 1%时，龙头可能已经涨停了

**盲区：**
- 一日游板块（消息刺激无持续性）会导致选到冲高回落的票
- **建议加分：** 板块连续2日涨幅 > 1%（持续性确认）

**量化实现：**
```
板块涨幅 = max(该股所属各概念板块今日涨幅)
通过条件 = 板块涨幅 > 1%
加分条件 = 板块连续涨幅天数 >= 2
```

---

### 4. 近5日有主力资金净流入 ⭐⭐⭐

**合理性：中等。** 主力资金数据在A股有参考价值，但数据本身有争议。

**实战要点：**
- 数据源：东方财富「个股资金流向」、同花顺「主力净流入」
- **核心问题：** 「主力资金」的定义各平台不同：
  - 东方财富：大单（>50万）净流入 - 大单净流出
  - 同花顺：特大单+大单之和
  - **事实：这些数据都是根据挂单金额推算的，不是真实的机构持仓变化**
- 5日净流入 > 0 只说明大单买入多，可能是机构建仓，也可能是游资短线操作

**盲区（最大争议点）：**
- 主力拆单：大机构会拆成小单规避监控，导致「净流入」失真
- 对倒：自买自卖制造流入假象
- **建议：** 不要只看5日净流入金额，要结合**流入天数/5 > 60%**（至少3天流入）和**净流入/流通市值 > 0.5%**（流入占比有意义）

**量化实现：**
```
5日主力净流入 > 0
AND 流入天数 >= 3（5天中至少3天净流入）
AND 累计净流入 / 流通市值 > 0.5%
```

---

### 5. 流通市值 50亿-300亿 ⭐⭐⭐⭐⭐

**合理性：非常高。** 这是精选中盘股的黄金区间，兼顾流动性和弹性。

**实战要点：**
- **50亿以下：** 流动性差，大资金进出困难，容易被操控
- **300亿以上：** 机构扎堆，波动率降低，短线弹性不足
- **50-300亿：** 游资和机构都偏好，容易产生龙头股
- 也可以细分：
  - 50-100亿：弹性最大，但风险较高
  - 100-200亿：最佳平衡区
  - 200-300亿：偏稳健

**量化实现：**
```
流通市值 = 最新价 × 流通股本
通过条件 = 50亿 <= 流通市值 <= 300亿
```

## 二、加分条件逐条拆解

### 6. 涨停基因（近1月有涨停） ⭐⭐⭐⭐

**实战价值：高。** 有涨停基因的票说明有活跃资金关注，股性活。
- 区分首板和连板：连板（2板以上）基因更珍贵
- 涨停后回调充分再启动效果最好
- **量化：** 近20个交易日内有 >= 1次涨停

### 7. 板块龙头或跟风强势 ⭐⭐⭐⭐

**实战价值：高。** 龙头溢价是A股最强的alpha来源。
- 龙头定义：板块内率先涨停/涨幅最大
- 跟风强势：板块涨幅前3名
- **量化难点：** 龙头判断需要实时排序，建议用「板块内今日涨幅排名 <= 3」

### 8. 政策直接受益标的 ⭐⭐⭐

**实战价值：中高，但量化困难。**
- 需要NLP解析新闻/公告，判断是否与个股相关
- **简化方案：** 预设政策关键词库（如「新能源」「半导体」「AI」等），匹配板块概念名称
- 或使用东方财富「题材概念」中近期热门概念

### 9. 缩量回调至均线获得支撑 ⭐⭐⭐⭐

**实战价值：高。** 这是经典的「缩量回踩买入」形态。
- 缩量 = 当日成交量 < 5日均量 × 0.7
- 回踩 = 最低价触及MA10或MA20（最低价 < MA20 × 1.02）
- 支撑 = 收盘价回到MA20上方（阳线或长下影线）
- **量化实现难度中等，效果很好**

### 10. MACD在0轴上方金叉 ⭐⭐⭐⭐

**实战价值：高。** 0轴上方金叉是经典多头信号。
- 金叉 = MACD柱线由负转正，或DIF上穿DEA
- 0轴上方 = DIF > 0，说明中长期趋势向上
- **注意：** 金叉确认有滞后性，等到确认时股价已经涨了一段
- **改进：** 关注「将金未金」（DIF即将上穿DEA），提前埋伏

### 11. 情绪高涨（涨停家数多） ⭐⭐⭐

**实战价值：中等。** 市场情绪是短线交易的环境变量。
- 涨停家数 > 50：情绪亢奋，短线容易赚钱
- 涨停家数 < 20：情绪低迷，亏钱效应强
- **但要注意：** 涨停家数是结果指标，用它选股有滞后性
- **改进方案：** 用涨停家数作为**大盘情绪过滤**（而非个股加分项），情绪差时直接不选

## 三、规则整体评价

### 优势
1. **逻辑自洽：** 大盘→板块→个股，从宏观到微观层层过滤，思路清晰
2. **风险控制意识：** 市值区间+趋势确认，过滤了大部分垃圾股
3. **有实战味道：** 不是纯技术派，加入了板块效应和资金面的考量

### 需要改进的地方
1. **缺少风控维度：** 没有考虑个股基本面（ST、退市风险、财务异常）
2. **缺少卖出规则：** 有选股没有卖出，体系不完整
3. **缺少仓位管理：** 选出来的股票买多少？
4. **部分条件有重叠：** MA20站稳 + MACD 0轴上方，趋势信号重复

### 关键改进建议
- **必加条件：** 排除ST/退市风险股、上市不满60天的新股、当日换手率 < 3%（流动性不足）
- **必加条件：** 前期涨幅过大（近20日涨幅 > 50%）的票要排除，追高风险极大

## 四、量化实施方案

### 数据源规划

| 数据 | 来源 | 接口 | 备注 |
|------|------|------|------|
| 大盘指数日K | Tushare | `index_daily` | 上证指数MA20斜率 |
| 个股日K | Tushare | `daily` | MA20、MACD、成交量 |
| 实时行情 | 东方财富 | `push2.eastmoney.com` | 最新价、涨跌幅 |
| 板块涨幅 | 东方财富 | `push2.eastmoney.com` | 概念/行业板块排行 |
| 主力资金 | 东方财富 | `push2.eastmoney.com` | 个股资金流向 |
| 流通市值 | Tushare | `daily_basic` | 或 `stock_basic` × 实时价 |
| 涨停记录 | Tushare | `limit_list_d` | 涨停/跌停统计 |
| 涨停家数 | 东方财富 | `push2.eastmoney.com` | 市场情绪指标 |

### 筛选流程（执行顺序很重要）

```
第1步：大盘环境判断
  ├── 上升/震荡 → 继续
  └── 下降 → 停止筛选，输出"大盘环境不佳"

第2步：基础过滤（全市场5000+只 → 约500只）
  ├── 排除ST / *ST / 退市风险
  ├── 排除上市 < 60天
  ├── 排除创业板注册制（20%涨跌幅）新股
  ├── 排除近20日涨幅 > 50%（追高风险）
  └── 流通市值 50-300亿

第3步：趋势确认（约500只 → 约150只）
  ├── 收盘价 > MA20 × 1.01（站稳20日线）
  ├── MA20连续3日向上
  └── 当日换手率 > 3%

第4步：板块与资金（约150只 → 约30只）
  ├── 所属最强概念板块今日涨幅 > 1%
  ├── 近5日主力净流入 > 0
  └── 5日中至少3天净流入

第5步：加分项评分（30只打分排序）
  ├── 涨停基因：+10分（近20日有涨停）
  ├── 板块排名：+15分（板块前3名）/ +10分（前10名）
  ├── MACD金叉：+10分（0轴上方）/ +5分（0轴下方）
  ├── 缩量回踩：+10分（近3日内出现缩量回踩MA20形态）
  ├── 市场情绪：+5分（涨停家数>50时全体加分）
  └── 政策概念：+8分（匹配当日热门概念关键词）

第6步：输出Top N
  ├── 按综合评分排序
  ├── 输出前10只，含评分明细
  └── 附带风控提醒
```

### 评分体系设计

| 项目 | 权重 | 说明 |
|------|------|------|
| 大盘环境 | 一票否决 | 下降市不选股 |
| 趋势强度 | 25分 | MA20偏离度 + MA20斜率 |
| 板块效应 | 20分 | 板块涨幅 + 板块内排名 |
| 资金面 | 20分 | 5日净流入金额/占比 |
| 技术形态 | 20分 | MACD + 缩量回踩 + 涨停基因 |
| 市场情绪 | 15分 | 涨停家数 + 连板效应 |

**总分100分，建议关注 >= 60分的标的。**

### 建议执行时间

- **盘后运行（15:30-16:00）：** 用收盘数据做完整筛选，准备次日标的池
- **早盘运行（09:30-09:45）：** 用集合竞价数据 + 昨日筛选结果做二次确认
- 盘中不推荐频繁运行（数据不稳定，容易追高）
